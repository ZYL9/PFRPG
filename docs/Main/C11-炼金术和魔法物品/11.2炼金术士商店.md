# 11.2炼金术士商店

## 魔法服务

### 购买信息标准费用

- 平常的新闻和流言：通常不会聊到这类信息，可能会让角色们去到理发店或最近的沙龙打听。也可能会在与付费客户的聊天中蹦出来。

- 有关秘密、贵族或皇家成员、危险物品的准确信息：通常不知道也不关心，同上。

- 关于特定的、重要的人物、地点的谣言、传说、历史和信息（和魔法、政治、当前任务无关的，非通用知识）：50-300金。可能需要查阅相关书籍。可免费提供。

- 关于特定的、重要的人物、地点的谣言、传说、历史和信息（和魔法、政治、当前任务有关的）：100-600金。可能需要查阅相关书籍。偶尔免费提供。

- 对鲜为人知的事情、秘密、神器或关于魔法、魔法物品或强大的施法者的危险信息进行广泛的研究或离散的探秘：1000-8000金。变化幅度大，具体取决于危险性与消耗的时间。

- 地脉地理信息：100金。节点300金。大部分炼金术士不会分享这些信息。

- 辨识草药：5-20金。有时免费。

- 阅读/解读神秘文字/符号：见雕纹。

- 对魔法、超自然力量进行简单咨询：25-100金。有时免费。

- 辨识魔法药水与卷轴：50-100金。有时免费。炼金术士可以在他鉴定物品之前提出购买这些物品。如果在鉴定之前出售，炼金术士将拒绝透露它们究竟有什么作用，但会支付公平的价格(市场价值的20-30%)。

- 辨识武器是否具有魔法特性：非魔法5-20金。通常免费。如果有魔法特性则会回答“是”，具体效应需额外付费鉴定。

- 辨识魔法物品的准确效应：100-600金。关于物品和/或如何使用的警告或详细信息将取决于提供鉴定的炼金术士是否慷慨，以及他是否喜欢问询的人。如何使用特定物品或其特殊信息可能需要额外支付100 - 600金。他可能会在确认物品之前提出购买该物品。如果在鉴定之前出售，炼金术士将拒绝透露它们究竟有什么作用，但会支付公平的价格(市场价值的20-30%)；如果炼金术士特别想要则会出到很好的价格(市场价值的40-50%)，甚至更高（80%，通常为60%）。

### 出售雕纹

炼金术士不会与任何人教授或讨论雕纹或符号魔法的细节，包括那些所谓的魔导师和召唤师。他们所知晓的秘密不能流传出去，尤其不能被那些艺术家或普通人知晓。然而这些多职业施法者乐意为其他人提供简单的雕纹来保护物品或人物。唯一的危险是，购买者必须给出自己的真名，这样炼金术士可以把你从雕纹的触发目标中剔除。如果给了假名，那么雕纹要么无法生效、要么会被购买者触发。

- 辨识雕纹真伪：10金。是否激活、是否有危险30金。失活、安全移除50-100金。

- 翻译魔法符号、符文或文字：简单翻译：每页50金。复杂翻译：每页200-500金。

价格可能相差20-50%，通常是更多而不是更少，我们可以假设炼金术士会复制任何他觉得有趣或有价值的东西。

注意：一个炼金术士永远不会用雕纹、符文或神秘符号进行解释！翻译是用普通语言和外行术语进行的准确解释。

### 制作并布置雕纹（均为6级强度）

警报：静默：300G；发声：200G；触发：500G

颜色：任意颜色50G

保护免于：魔法、能量、隐形、亡灵/吸血鬼：每个1000G；魔能汲取：8000G；其他：每个500G

保护并造成：死亡、剧痛、目盲、燃烧、能量、隐形、火焰：每个2400G；其他：每个1200G

注：炼金术士很少会出售造成类雕纹或多重雕纹，且永远不会销售永久、威能或魔能汲取类雕纹。（当然特殊情况会使炼金术士打破规则，若真如此也会极为昂贵，可能会用各种强大的物品或知识来交易）。魔导师也会提供类似的服务并且还便宜10-30%。某些强大的雕纹会比普通的贵2-4倍。

### 出售法阵

炼金术士可以教导召唤师，或者极少情况下- 魔导师，如何制作法阵，但也只会教导这两种神秘技艺的实践者。炼金术士会通过询问有关威能词的问题来判断其是否是这两类施法者。那些胆敢欺骗炼金术士的人，会发现他们将变成炼金术士的实验材料了。正常情况下，炼金术士同时只会教导一个人。炼金术士也并不是了解所有的法阵，他也会专注于某些研究领域，通常也只了解1D6个力量法阵。

#### 法阵费用

保护法阵：3000-6000G

召唤法阵：6000-12000G

力量法阵（只会教给目标）：70000-150000G； dimensional Rift, teleport, power, death, power matrix and wonder 每个200000-250000G

## 出售法术教学

大部分炼金术士（并非全部）会教导法师新的法术。只会教导能证明自己的法师（展示魔法并且回答一些法师的知识），学习的法术每有1级就需要2天来学习。因此学习一个2级法术需要4天时间。

**限制与状态：**法术教学由炼金术士本人或其法师助理进行，通常会在一个由其选择的地方进行教学活动，通常会远离其他人。在教学时间内，学徒只会和老师待在一起，期间的食物消耗也都由老师提供。如果对此有任何异议，那么教学会立刻终止。有侮辱性或含沙射影者，课程也将被取消，恕不退款。所有法师都理解并接受这个不成文的规则。老师必须知道这个法术才能教授它，所以所有的法术，即使是很低级的法术，也可能无法从炼金术士那里获得。

**保证：**保证教导的法术是真实有效的，能完美运作。

**法术费用：**炼金术士可以教导任何等级的法术（无仪式），但大部分炼金术士拒绝教导7/8级以上的法术。价格可能听上去过高，但毕竟法术你一旦学会，你终生可用。

1级：5000G

2级：10000G

3-4级：20000G，有时有10%溢价

5-6级：35000G，有时有20-50%溢价

7-8级：50000G，有时有20-50%溢价

9-15级及传说法术永不提供。如果真的要提供，推荐价格为：100000-500000G+

注：进攻性法术通常会贵10-30%，如迷惑之缕、火球术、呼召雷霆、驱散魔法造物。维度法术和强大的变形法术，包括各种传送术、时跃、融影咒、各种隐形术和变形术都会贵30-50%。

## 出售魔法卷轴

炼金术士是钯界之中少数可以将法术转化为魔法卷轴的人物。通常来说“法术卷轴”就是“魔法卷轴”，是一张卷起来的(从未折叠过的)羊皮纸，上面有1 -3个不同的法术所转换成的魔法卷轴。任何可以读懂卷轴书写语言的人（通常是精灵语）可以阅读并激活卷轴之上的法术。一旦卷轴被阅读，魔法将被激活并施放。施放法术所需的潜能点已被注入并存储在卷轴之上的文字之间。因此在阅读完卷轴后，其上文字将消散，法术被施放。如果读到一半停止，超过3秒，法术将失效，卷轴其上的文字仍会消散，即便你并没有放出法术。

在阅读后卷轴上只会残留法术的名称，若以阅读卷轴标题是没有问题的。不要过度展开卷轴，以避免不小心看到注魔的词句并阅读它们。如果一个卷轴记录了不止一个法术，那么其制作者通常会写从上到下依次书写，并在每个法术之间留足空位和单独的标题。

注：使用银制符文来书写卷轴也无法使卷轴永久保存。两种不同类型的魔法技艺无法作用于同一张羊皮纸上。

限制：仅能使用一次，只能以一种语言书写。一旦激活/阅读，法术将消失。如果一个人无法阅读，也不懂卷轴上书写的语言，那么其就无法施放。展开卷轴并假装或试图阅读可能会导致魔法在施放失败的情况下消失。

保证：无论炼金术士如何声明（6级，14豁免，特定法术等），保证真实可用，或可以退款。

魔法卷轴豁免：13/14+

魔法卷轴强度：效力，持续时间，范围和伤害可以与炼金术士自己作为施法者的个人经验等级一样高(通常是6 - 12级)。然而，卷轴上的大多数咒语都是由4、5或6级法术施放;炼金术士可以准确地指出魔法的等效经验等级(随机选择1D4+3)。一个等效为8-10级法师强度的卷轴通常会贵1D6×1000G，无论其法术等级几何。比如一个2级法术卷轴，如果等效10级法师强度，那么通常为800-1200G+1000-6000G。

法术费用：任何等级的法术都可以转化为卷轴，但8级以下的法术更常见。

1-2级：800-1200G

3-5级：2000-2500G，伊森之甲、融影咒、逃脱术和透特之眼通常是顶格的价格（3000G），有时有10-50%溢价

6-7级：3000-4000G，有时有10-50%溢价

8-9级：5000-7000G，有时有20-50%溢价

10-12级：10000-15000G，有时有20-50%溢价

13-15级：200000-30000G，有时有30-100%溢价

传奇法术：50000-100000G。极其稀有，根据具体情况、销售者和购买者可能会是所列价格的两到三倍。

注：卷轴的成本可能比上面列出的平均价格少20%，多100%。这在很大程度上取决于卷轴在该地区的供应情况、追逐者的绝望程度以及卖家的慷慨程度。不幸的是，大多数炼金术士都是自私和邪恶阵营，并以利用局势为自己的利益而臭名昭著。

炼金术士通常都是喜怒无常、爱评判和古怪的。赢得对方的好感，你通常会得到公平的对待（大多数情况下），但如果惹恼了对方，就做好付出高昂代价或者受到粗鲁对待的准备。此外，种族、阶级和职业偏见也可能导致价格变动，或者增加或减少可购买的卷轴和魔法物品的范围。也就是说，一个讨厌矮人的（狼人、人类、贵族等）炼金术士可能会告诉矮人顾客他只有少量卷轴可用（实际上他有十倍之多），而且价格很可能比正常高出至少20%-50%。相比之下，当一位精灵或同行询问时，精灵炼金术士会有大量普通或低于市场价的物品可供选择。任何在魔法商店出售的商品都是如此。

可得性：之前列出的价格是已经制作完成并准备出售的卷轴的价格。这限制了可供购买的卷轴的种类，仅限于该地区最受欢迎的卷轴，以及偶尔出现的特定魔法或稀有魔法卷轴。大多数炼金术士通常会准备以下1D4种常用卷轴中的每一种：治愈、中和毒素、巧言术、看破隐形、简易隐形术、感知魔法和火球术。

炼金术士恰好拥有角色所需物品的概率完全就是一场碰运气的游戏。一个简单的经验法则是，对于每项请求都掷D100，但可能有例外（即使上面列出的物品也可能缺货）。对于1-7级法术卷轴，有40%的概率有一到两张卷轴上有该法术。对于8-12级法术卷轴，只有25%的概率，而13级及以上，只有12%的概率有该法术的卷轴可用。记住，最终还是由GM控制物品是否可用，所以如果他或她不想让PC得到这类物品，那么就不会有。

定制卷轴：如果炼金术士有时间和意愿，他可以创造出角色想要的任何卷轴。想要购买定制卷轴，需要至少24小时(大多数人要求48小时到一周)的时间来制作一个卷轴，并且卷轴上的每个法术都比正常最高价格还要高50-100%。无法定制10级以上的法术。定制卷轴可以用买方要求的任何语言书写。

获取法术卷轴的其他方法：一些高级牧师也可以创造卷轴，可以卖出这些卷轴来支持教会，或者作为奖励或防御手段给予教会的捍卫者或光明斗士。卷轴也可以由皇室、牧师或神灵授予，也可以在地下城、古代废墟、寺庙、邪恶的施法者、龙和恶人的巢穴中被捕获、窃取或发现。
